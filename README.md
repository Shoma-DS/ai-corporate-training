# AI法人研修

AI法人研修の資料、原稿、スライド、配布資料、調査メモを作るための公開作業版リポジトリです。

## 公開前提の運用

このリポジトリは public です。次の情報はコミットしないでください。

- 顧客名、参加者名、個人情報
- 未公開の事業情報、社内資料、契約情報
- API キー、認証情報、`.env` ファイル
- 外部公開の許諾がない画像、記事、動画、スライド

顧客固有の一時メモや非公開素材は `private/` に置きます。`private/` は Git 管理対象外です。

## ディレクトリ

- `drafts/`: 講義原稿、ナレーション、記事化前の下書き
- `slides/`: スライドのソース、構成案
- `handouts/`: ワークシート、チェックリスト、配布資料
- `research/`: 調査メモ、出典整理、事例メモ
- `templates/`: 研修企画、セッション設計、教材作成のテンプレート
- `assets/`: 公開可能な画像、図版、アイコン、ライセンスメモ
- `scripts/`: 変換、検証、自動化スクリプト
- `exports/`: PDF や配布用ファイルなどの生成物

## 作業フロー

1. `templates/training-brief.md` から研修企画を作る。
2. `templates/session-outline.md` で各回の目的、流れ、演習を整理する。
3. 原稿は `drafts/`、スライドは `slides/`、配布資料は `handouts/` に分けて作る。
4. 出典や参考リンクは `research/` に残す。
5. 公開前に、機密情報と権利未確認素材が混ざっていないか確認する。
