# P3 Upload Bundles

Objective: Build one Canva import bundle per session.

Do:
- Use the local slide inventory from P1.
- Prefer a PPTX bundle where each `スライド画像/Sxx.png` is placed as one full-slide page.
- Save bundles outside `非公開/` unless they contain private data; generated public-safe slide images are acceptable.
- Record bundle paths privately.

Do not:
- Modify slide image files.
- Run Canva MCP `_image_to_design` for every slide.
- Add Canva URLs to public repo files.

Expected output:
- Six session-level upload bundles.

Verification:
- Bundle page counts are 42/40/44/40/40/36.
- Page order follows S01, S02, S03...
