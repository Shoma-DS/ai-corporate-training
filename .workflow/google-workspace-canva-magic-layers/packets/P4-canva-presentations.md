# P4 Canva Image-based Presentations

Objective: Create one image-based Canva presentation per session and move it to the matching Canva folder.

Do:
- Upload each session bundle or slide image set to Canva.
- Create one multi-page Canva presentation per session.
- Move each presentation to its matching session folder.
- Record `design_id`, `edit_url`, `view_url`, `folder_id`, and `page_count` only in private logs.

Do not:
- Convert every slide to Magic Layers.
- Store real Canva URLs or design IDs in tracked public files.

Expected output:
- Six Canva presentations, one per session.
- Private URL/index rows for each session.

Verification:
- Each presentation page count matches the local session image count.
- Each presentation is in the expected Canva folder.
