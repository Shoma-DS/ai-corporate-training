## 情報量増強の主な変更点

### 元スライドとの比較
| 観点 | 元スライド | 新スライド案 |
|---|---|---|
| 1スライドの構造 | 見出し＋アイコン＋数行テキスト | 結論型ヘッドライン＋表or箇条書き3〜6ブロック |
| 見出しスタイル | 説明型（「データドリブン思考とは？」） | So What型（「データドリブン思考は...5ステップを継続して回すことで機能する」） |
| 具体例 | 抽象的・汎用的 | 業種別（製造・小売・サービス・物流等）の具体事例 |
| 比較表 | 少ない | 各章に比較表を配置（定量的な差異を明示） |
| 演習 | なし | 各章末に10分ワーク（合計第1回4回・第2回3回） |
| 数値 | ほぼなし | 効果の参考数値を追加（「想定例」明記） |
