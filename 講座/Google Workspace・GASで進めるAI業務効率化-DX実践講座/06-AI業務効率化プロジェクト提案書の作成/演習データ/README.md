# 第6回 演習データ

第6回は、導入提案書を作る回です。提案書の材料、KPI、ロードマップ、リスク対策、レビュー観点だけを置きます。

| ファイル | 用途 |
| --- | --- |
| `提案書材料_プロトタイプ一覧.csv` | これまで作ったプロトタイプを提案書に整理する |
| `KPI効果試算サンプル.csv` | 削減時間、対応速度、確認漏れ、再利用率を試算する |
| `導入ロードマップサンプル.csv` | 試行、本番準備、教育、定着、改善の段階を作る |
| `リスク対策サンプル.csv` | AI入力禁止、権限、実行制限、停止時復旧を提案書に入れる |
| `発表レビュー観点.csv` | 提案書の自己レビューと講師例との比較に使う |
