講座タイトル: 生成AI・GASで実践する業務変革・DX推進講座。

## この台本の使い方

- この台本は**画面録画による動画講座**用です。ライブ配信ではありません。受講者は動画を視聴して学習します。
- 「読み上げ」は、講師が録画時にそのまま読み上げる文です。
- 「スライド切替」は、切り替えるスライド番号とタイトルです。
- 「画面共有」は、ダミー環境の操作画面を見せるタイミングです。手順・時間・ポイントを記載しています。
- 「ワーク指示」は、受講者が動画を一時停止して `ワークシート.md` に書く演習です。
- 「講師メモ」は読み上げず、録画時の注意喚起に使います。
- 実在会社名、顧客名、価格、連絡先、個人情報、社内固有情報は扱いません。画面共有は必ずダミーデータか公式公開画面を使います。

## 演習用サンプルコード

講師メモ:
このコードは研修用のダミーシートを前提にしています。本番利用では、権限、共有範囲、データ件数、実行時間、通知条件、復旧手順を確認してから使います。

```javascript
const SOURCE_SHEET = '売上CSV';
const TARGET_SHEET = '未請求一覧';
const LOG_SHEET = '処理ログ';

function onOpen() {
  SpreadsheetApp.getUi()
    .createMenu('研修メニュー')
    .addItem('未請求を抽出', 'extractUnbilledRows')
    .addToUi();
}

function extractUnbilledRows() {
  const ss = SpreadsheetApp.getActive();
  const source = ss.getSheetByName(SOURCE_SHEET);
  const target = ss.getSheetByName(TARGET_SHEET) || ss.insertSheet(TARGET_SHEET);
  const log = ss.getSheetByName(LOG_SHEET) || ss.insertSheet(LOG_SHEET);

  if (!source) {
    throw new Error(`元シートが見つかりません: ${SOURCE_SHEET}`);
  }

  const values = source.getDataRange().getValues();
  if (values.length < 2) {
    writeLog_(log, '未請求抽出', 0, 0, 'データ行なし');
    return;
  }

  const headers = values[0];
  const rows = values.slice(1);
  const headerMap = Object.fromEntries(headers.map((name, index) => [name, index]));

  const requiredHeaders = [
    '取込日',
    '伝票番号',
    '顧客ID',
    'カテゴリ',
    '商品名',
    '数量',
    '単価',
    '金額文字列',
    '担当者',
    '対応状況',
  ];
  const missingHeaders = requiredHeaders.filter((name) => headerMap[name] === undefined);
  if (missingHeaders.length > 0) {
    throw new Error(`必要な列がありません: ${missingHeaders.join(', ')}`);
  }

  const outputHeaders = headers.concat(['チェック結果']);
  const outputRows = [outputHeaders];
  const seenSlipNumbers = new Set();
  let duplicateCount = 0;

  for (const row of rows) {
    const slipNumber = row[headerMap['伝票番号']];
    const status = row[headerMap['対応状況']];
    const isDuplicate = seenSlipNumbers.has(slipNumber);

    if (isDuplicate) {
      duplicateCount += 1;
    }
    seenSlipNumbers.add(slipNumber);

    if (status === '未請求' || status === '重複疑い') {
      const checkResult = isDuplicate || status === '重複疑い' ? '要確認' : '抽出対象';
      outputRows.push(row.concat([checkResult]));
    }
  }

  target.clearContents();
  target.getRange(1, 1, outputRows.length, outputRows[0].length).setValues(outputRows);

  const extractedCount = outputRows.length - 1;
  writeLog_(log, '未請求抽出', extractedCount, duplicateCount, '完了');
  ss.toast(`${extractedCount}件を抽出しました`, '研修メニュー', 5);
}

function writeLog_(logSheet, processName, extractedCount, duplicateCount, result) {
  if (logSheet.getLastRow() === 0) {
    logSheet.appendRow(['実行日時', '処理名', '抽出件数', '重複件数', '結果']);
  }
  logSheet.appendRow([new Date(), processName, extractedCount, duplicateCount, result]);
}
```
