# 演習データ

このフォルダは、Google Workspace・GASで進めるAI業務効率化/DX実践講座。の録画、デモ、演習で使うダミーデータ置き場です。

すべて架空データです。実在する顧客、社員、会社、メールアドレス、契約情報、問い合わせ本文は入れません。

## ファイル

| ファイル | 用途 |
| --- | --- |
| `問い合わせ受付データ.csv` | Forms受付、問い合わせ分類Gem、返信案、担当者通知 |
| `申請承認データ.csv` | 申請フォーム、承認台帳、期限通知、帳票作成 |
| `日報データ.csv` | 日報集約、週次要約、未提出確認、Docs報告書 |
| `売上CSV整形データ.csv` | CSV整形、重複チェック、条件抽出、集計 |
| `タスク期限管理データ.csv` | 期限管理、Calendar/Gmail連携、リマインド |
| `顧客フォロー架空データ.csv` | 顧客フォロー、メール案、次回予定 |
| `ダミー議事録メモ.md` | Drive/Docs要約、会議後タスク抽出 |
| `Meet文字起こしサンプル.md` | Meet文字起こしの要約、決定事項・宿題抽出、Docs議事録整形 |
| `Meetアクション台帳サンプル.csv` | 文字起こしから抽出したアクションのSheets台帳化、担当者別リマインド |

## 使い方

1. Googleスプレッドシートで新規ファイルを作る。
2. `ファイル > インポート > アップロード` でCSVを取り込む。
3. 取り込み後、必要に応じてシート名を日本語に変更する。
4. デモ録画では、個人アカウント名、個人メール、実データが映らない検証用環境で使う。

## Meet文字起こし演習

1. `Meet文字起こしサンプル.md` をGeminiまたはGemに入力し、会議要約、決定事項、未決事項、宿題を抽出する。
2. 抽出結果を `Meetアクション台帳サンプル.csv` の列に合わせて整える。
3. Sheetsに取り込み、担当者、期限、ステータス、優先度でフィルタや条件付き書式を設定する。
4. GASで担当者別アクション一覧、期限前日・当日のリマインド、Docs議事録の整形を行う。
5. 本番データを扱う場合は、個人情報、顧客情報、社内固有情報を削除または匿名化し、AI出力を人が確認してから確定する。
