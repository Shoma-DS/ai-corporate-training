# 第2回 演習データ

第2回は、Forms回答とSheets台帳を設計する回です。フォーム回答、台帳列定義、担当者マスタだけを使います。

| ファイル | 用途 |
| --- | --- |
| `問い合わせフォーム回答サンプル.csv` | 問い合わせ受付フォームの設問と回答先シートを確認する |
| `申請フォーム回答サンプル.csv` | 申請・承認台帳の列、承認期限、添付有無を設計する |
| `日報フォーム回答サンプル.csv` | 日報の集計しやすい列設計と上長確認列を考える |
| `台帳列定義サンプル.csv` | 回答シートと管理台帳を分けるための列定義を作る |
| `担当者マスタサンプル.csv` | 入力規則、担当候補、通知対象のマスタとして使う |
