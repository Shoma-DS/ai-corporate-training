# 演習データREADME

このフォルダのデータは、社内ナレッジ活用・根拠確認DX推進講座用のダミーデータです。

## ファイル

- `サンプル商事_資料一覧.csv`: NotebookLMに入れる資料、入れない資料を判断するための資料棚卸しサンプル。
- `サンプル商事_FAQ素材.md`: FAQ化と根拠付き回答の演習に使う架空マニュアル。
- `サンプル商事_会議メモ.md`: 会議後のアクション整理に使う架空議事録。

## 利用上の注意

- 実在企業、実在顧客、実在社員の情報は含めていない。
- 実務で利用する場合は、各社の情報管理ルールに従い、必要に応じて匿名化、抜粋化、承認取得を行う。
- AI出力を業務に使う前に、根拠資料、最新版、担当者確認を行う。
