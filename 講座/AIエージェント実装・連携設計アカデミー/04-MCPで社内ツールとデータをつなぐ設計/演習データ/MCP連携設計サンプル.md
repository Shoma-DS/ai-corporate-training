# MCP連携設計サンプル

## 対象業務

問い合わせ一次対応の分類候補とFAQ参照。

## Resources

- 社内FAQ Markdown
- 問い合わせカテゴリ定義
- 返信時の確認チェックリスト

## Prompts

- 問い合わせ本文をカテゴリ候補、優先度候補、確認メモに分ける手順
- 回答案を作る前に、根拠と不明点を分ける手順

## Tools

- FAQ検索: 読み取り専用
- チケット下書き作成: 書き込み。人の承認が必要

## 接続しない情報・操作

- 実在顧客データ
- 契約情報
- 外部送信
- 削除

## ログと停止

- 検索ログと下書き作成ログを残す。
- 問題があればMCP設定を無効化し、手動対応に戻す。
