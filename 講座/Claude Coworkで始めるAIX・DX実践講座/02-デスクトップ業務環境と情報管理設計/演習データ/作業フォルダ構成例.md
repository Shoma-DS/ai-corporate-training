# 作業フォルダ構成例

```text
AIX-DX試行/
  00-ルール/
    作業ルール.md
    アクセス範囲.md
  10-入力/
    週次レポート元データ.csv
    会議メモサンプル.md
  20-作業中/
    AI下書き/
  30-出力/
    週次レポート下書き.md
    確認観点.md
  40-レビュー済み/
    週次レポート_レビュー済み.md
  90-削除候補/
    削除候補一覧.md
```

## ルール

- `10-入力/` にはダミーデータだけを置く。
- `20-作業中/` にはAI下書きを置く。
- `30-出力/` は提出候補であり、未レビューの成果物を置く。
- `40-レビュー済み/` は人が確認したものだけを置く。
- `90-削除候補/` は候補一覧だけを置き、実ファイルは削除しない。
