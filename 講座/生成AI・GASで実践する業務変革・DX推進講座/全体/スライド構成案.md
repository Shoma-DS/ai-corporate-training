# 生成AI・GASで実践する業務変革・DX推進講座。 スライド構成案

## Deck Concept

中小企業がすでに使っているGoogle Workspaceを土台に、Googleスプレッドシート、Googleフォーム、Google Meet、GAS、無料範囲で使えるGemini/Gem、利用環境に応じたGeminiのGoogle Workspace連携を組み合わせ、Excel的な転記・集計・チェック・通知・会議後の宿題整理を減らす講座。

講座全体は「業務棚卸し -> フォーム/台帳設計 -> Sheets自動処理 -> Gem/Gemini活用 -> AI/GAS運用設計 -> 提案書作成」で進める。

課金や管理者設定がないと説明しづらいGemini for Google Workspace高度機能、Deep Research、AI Studio、Workspace Studioは必須演習に入れない。一方で、Geminiアプリ、Gem、利用環境に応じたGeminiのGoogle Workspace連携、Meet文字起こしは本編に入れ、分類、要約、返信案、GASコード読解、Gmail/Drive/Docs/Calendar/Meet文字起こしの情報整理、業務プロンプトの共有まで扱う。

サービス紹介、操作説明、作業風景を見せるスライドでは、抽象図解だけで表現しない。公式ロゴ、公式公開スクリーンショット、またはダミー環境のスクリーンショットを配置する。画像生成AIに実在ロゴや実在サービス画面を描かせない。
