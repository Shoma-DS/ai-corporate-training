# 第4回 演習データ

第4回は、Gem/Geminiで分類、要約、文書整理を行う回です。AI入力用の少量データと、出力レビュー用データだけを置きます。

| ファイル | 用途 |
| --- | --- |
| `問い合わせAI分類入力.csv` | 問い合わせ本文をカテゴリ、優先度、返信方針に分類する |
| `日報要約入力.csv` | 日報を部署別課題、次アクション、確認事項に要約する |
| `Drive資料要約サンプル.md` | Drive/Docs資料要約の代替入力として使う |
| `Meet文字起こしサンプル.md` | Meet文字起こしの代替入力として使う |
| `Meetアクション台帳サンプル.csv` | AI抽出結果を台帳列へ戻せるか確認する |
| `AI出力レビュー練習.csv` | AI出力を事実、推測、要確認、利用不可に分ける |
