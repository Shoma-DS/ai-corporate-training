# 第3回 演習データ

第3回は、GASで表処理を自動化する回です。CSV取込、未対応抽出、重複チェック、期限リマインドに必要なCSVだけを置きます。

| ファイル | 用途 |
| --- | --- |
| `CSV取込_売上明細_5月1週.csv` | 1つ目の取込CSV。ヘッダー、金額文字列、未請求行を確認する |
| `CSV取込_売上明細_5月2週.csv` | 2つ目の取込CSV。重複、担当者空欄、カテゴリ表記ゆれを確認する |
| `問い合わせ処理台帳_GAS演習.csv` | 未対応抽出、担当者別転記、ステータス更新、ログ追記に使う |
| `タスク期限リマインド_GAS演習.csv` | 期限前日・当日の未完了タスクを担当者別にまとめる |

フォーム設計やAI要約の練習データは、この回には置きません。
