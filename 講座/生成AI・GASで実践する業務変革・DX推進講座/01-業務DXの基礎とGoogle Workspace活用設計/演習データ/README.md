# 第1回 演習データ

第1回は、業務DXの対象を選ぶ回です。ここには業務棚卸し、改善テーマ選定、手作業の課題観察に使うデータだけを置きます。

| ファイル | 用途 |
| --- | --- |
| `業務棚卸しサンプル.csv` | 業務を入力・処理・出力・確認に分解し、AI/GAS/人の役割分担を考える |
| `改善テーマ評価サンプル.csv` | 効果、難易度、情報リスク、データ整備状況で優先順位をつける |
| `CSV手作業観察サンプル.csv` | コピー、整形、重複確認、未対応抽出がなぜ負担になるかを観察する |

他回で使うフォーム回答、GAS処理、AI分類、Meet文字起こしデータは置きません。
