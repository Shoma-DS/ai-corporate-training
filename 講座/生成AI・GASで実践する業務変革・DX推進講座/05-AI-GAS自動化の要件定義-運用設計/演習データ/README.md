# 第5回 演習データ

第5回は、AI/GAS自動化の要件定義と運用設計の回です。実装用ではなく、設計・リスク・テストに使うデータだけを置きます。

| ファイル | 用途 |
| --- | --- |
| `要件定義ケース_問い合わせ通知.csv` | 対象業務、入力元、処理、出力、非対象を整理する |
| `運用リスクチェックケース.csv` | 権限、AI入力禁止、実行制限、通知漏れ、復旧を確認する |
| `テストケースサンプル.csv` | 正常系、空欄、重複、権限ミス、AI誤分類などのテスト観点 |
| `Meet文字起こし入力サンプル.md` | 文字起こし入力元の利用可否と代替運用を設計する |
| `Meetアクション台帳_運用設計.csv` | 会議アクション台帳の通知・復旧・確認ルールを設計する |
| `権限保存設計サンプル.csv` | Drive保存先、閲覧者、編集者、外部共有可否を整理する |
