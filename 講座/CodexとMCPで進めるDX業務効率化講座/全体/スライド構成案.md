# スライド構成案

講座名: CodexとMCPで進めるDX業務効率化講座

## Deck Concept

業務課題をリポジトリ化し、Codexで改善プロトタイプを作り、MCPで社内ツールやデータとの接続を安全に設計する。講座全体は「業務課題整理 -> 作業環境設計 -> プロトタイプ実装 -> MCP連携設計 -> 評価・運用 -> 提案書作成」で進める。

採用テンプレート: `isometric-corporate-clean`

採用理由:

- DX、AI、業務効率化、運用設計、成果物を整理する法人研修に合う。
- 公式ロゴ、画面共有、スクリーンショットと図解を併用しやすい。
- 高度デジタル人材向けでも、白背景・ネイビー/ティール基調で実務的に見せられる。

## 素材方針

- Codex、OpenAI、MCPのロゴや画面を使う場合は、公式サイトまたは公式ブランドガイドラインから取得し、`素材/ロゴ/` と `全体/調査/` に取得元を残す。
- 画像生成AIに実在ロゴや実在UIを描かせない。
- 操作説明はダミーリポジトリ、架空データ、公式公開画面、またはダミー環境のスクリーンショットを使う。
