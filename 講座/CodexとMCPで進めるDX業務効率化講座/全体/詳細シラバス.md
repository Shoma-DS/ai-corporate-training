# 詳細シラバス

講座名: CodexとMCPで進めるDX業務効率化講座

## 講座の位置づけ

本講座は、生成AIエージェントを業務改善の実装・検証・運用設計へ組み込むための高度デジタル人材向け講座である。CodexとMCPを中心に扱うが、講座の主役はツール操作ではなく、業務課題整理、要件定義、プロトタイプ、連携設計、リスク管理、導入提案である。

## 評価観点

- 業務課題を、実装前にAs-Is/To-Beと要件に分解できている。
- Codexに依頼する内容が、目的、制約、期待出力、検証方法まで具体化されている。
- ダミーデータ、公開不可情報、秘密情報、承認が必要な操作を区別できている。
- MCP連携で、Resources、Prompts、Tools、認証、承認、ログ、停止手順を説明できる。
- プロトタイプの結果を鵜呑みにせず、テスト、レビュー、リスク確認を行っている。
- 導入提案書で、効果、リスク、運用、ロードマップ、次アクションを説明できる。
