# 第6回 画像生成プロンプト

採用テンプレートID: `isometric-corporate-clean`

## 共通ルール

- 16:9、法人研修スライド向け、余白広め、文字を置けるスペースを確保する。
- 実在ロゴ、実在UI、特定企業名、個人情報、価格、申込先、助成率は描かない。
- CodexやMCPは、抽象化した開発環境、接続ノード、承認フローとして表現する。
- 画面共有スライドは、実ファイルを後で表示する前提のシンプルな導入ビジュアルにする。
