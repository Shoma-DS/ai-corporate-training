# 第2回 スライド構成案: 業務リポジトリと安全な作業環境の設計

対象講座: CodexとMCPで進めるDX業務効率化講座

## ねらい

第1回のCodexタスクブリーフを、Codexが安全に作業できる業務改善リポジトリへ落とし込む。公開不可情報、ダミーデータ、AGENTS.md、Git差分、サンドボックス、承認の考え方を扱う。
