# 第2回 演習ガイド

## 目的

Codexに作業を依頼する前に、業務改善リポジトリ、作業ルール、ダミーデータ、安全チェックを整える。

## 手順

1. `業務リポジトリ構成例.md` を確認する。
2. 自分の改善テーマに必要なフォルダとファイルを書く。
3. `AGENTSひな形.md` を参考に、作業ルールを作る。
4. `公開不可情報チェックサンプル.csv` で、AI入力や公開資料に入れない情報を確認する。
5. 作業安全チェックを完成させる。

## 完成条件

- Codexが触ってよい範囲が書かれている。
- 実データを使わない検証方法が書かれている。
- 承認が必要な操作が分かれている。
