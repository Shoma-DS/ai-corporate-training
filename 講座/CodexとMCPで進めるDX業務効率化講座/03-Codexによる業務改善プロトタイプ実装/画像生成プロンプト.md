# 第3回 画像生成プロンプト

採用テンプレートID: `isometric-corporate-clean`

## 共通設定

```text
Create a 16:9 Japanese corporate training slide, clean white background, navy and teal accent palette, light gray thin card borders, subtle soft shadows, crisp isometric vector infographic style, organized business presentation layout, professional DX and AI training atmosphere, no real brand logos, no real UI screenshots, no confidential information.
```

共通ネガティブプロンプト:

```text
Avoid fake product screenshots, fake OpenAI logos, fake MCP logos, API keys, emails, phone numbers, real company names, real personal data, pricing, dense code text, dark cyber style, decorative gradient blobs, tiny unreadable Japanese text.
```
