# 第3回 講師台本: Codexによる業務改善プロトタイプ実装

講座名: CodexとMCPで進めるDX業務効率化講座

講師メモ:
この台本は録画動画によるeラーニング用。ライブ共有、発表、相互レビューは行わない。実演はダミーリポジトリとダミーデータで行い、外部送信、削除、実データ利用はしない。

講師メモ:
準備物は、第3回スライドS01-S40、`ワークシート.md`、`配布資料/演習ガイド.md`、`演習データ/問い合わせ対応サンプル.csv`、`演習データ/週次レポート元データ.csv`、`演習データ/期待出力例.md`。

スライド切替:
S01「Codexによる業務改善プロトタイプ実装」

読み上げ:
「第3回では、Codexを使って業務改善プロトタイプを作ります。ここでの目的は、大きなシステムを一気に作ることではありません。1つの入力を受け取り、1つの処理を行い、1つの出力を作る。まずこの小さな範囲で、業務に使えるかを検証します。」

スライド切替:
S03「プロトタイプは小さく作る」

読み上げ:
「プロトタイプは小さく作ります。最初から画面、通知、権限、外部連携まで全部入れると、何がうまくいって何が危ないのか判断できません。工場で新しい機械を入れる前に、まず材料の置き場所と作業手順を決めるように、AIエージェントの実装も小さな流れから始めます。」
